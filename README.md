## 🚀 How to Contribute
1. **Star the Repo**
2. **Follow If You Want**
3.  **Fork the repository**  
4. **Clone your fork**  
5. **Create a branch**  
6. **Make your changes**  
7. **Commit and push**  
8. **Create a Pull Request**

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed instructions.
